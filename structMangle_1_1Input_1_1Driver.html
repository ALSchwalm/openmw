Detailed Description

Input::Driver is the main interface to any input system that handles keyboard and/or mouse input, along with any other input source like joysticks.

It is really a generalized event system, and could also be used for non-input related events. The definition of the event codes and structures are entirely dependent on the implementation.

A system-independent key code list will be found in keys.hpp, and input drivers should privide optional translations to/from this list for full compatibility.
