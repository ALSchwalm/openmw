Detailed Description

Manual resource loader for NIF meshes. This is the main class responsible for translating the internal NIF mesh structure into something Ogre can use.

You have to insert meshes manually into Ogre like this:

NIFLoader::load("somemesh.nif");

This returns a list of meshes used by the model, as well as the names of their parent nodes (as they pertain to the skeleton, which is optionally returned in the second argument if it exists).
